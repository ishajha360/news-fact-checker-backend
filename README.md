# News Fact Checker - Backend

REST API backend for news credibility analysis using Spring Boot and Groq AI.

## Tech Stack
- Java, Spring Boot
- Groq AI API
- MVC Architecture

## How to Run
1. Add your API key in application.properties
2. Run NewsanalyzerApplication.java
